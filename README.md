# Kerr Black-Hole Superradiance and Variational Quantum Simulation

This repository accompanies an MSc research project on Kerr black-hole superradiance and variational quantum simulation.

## Main Contributions
- Finite-difference computation of Kerr scalar-field quasi-bound states
- Numerical investigation of superradiant instability
- Schrödingerisation of a non-Hermitian radial operator
- Hermitian embedding into a quantum-compatible Hamiltonian
- Pauli decomposition into qubit operators
- Variational Quantum Eigensolver (VQE) implementation
- End-to-end Kerr-to-VQE proof-of-concept pipeline

## Main Result
We establish a complete end-to-end pipeline from the Kerr scalar-field eigenvalue problem to a Variational Quantum Eigensolver implementation, demonstrating that the essential spectral structure of quasi-bound and superradiant modes survives dimensional reduction to a small quantum-compatible qubit representation.
